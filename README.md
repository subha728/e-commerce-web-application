E-Commerce Web Application

Overview

A full-stack E-Commerce Web Application built using the MERN stack. Users can register, log in, browse products, add items to cart, place orders, and view order history.

Features

- User Registration & Login
- JWT Authentication
- Product Listing
- Add to Cart
- Place Orders
- View Orders

Tech Stack

- HTML, CSS, JavaScript
- Node.js, Express.js
- MongoDB Atlas
- JWT Authentication
